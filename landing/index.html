How It Works

Download, install, and start coding with AI in under 5 minutes.

1

Download & Install

Download the app for your OS from the download section below, or build from source with pnpm tauri dev.

2

Set Up AI (choose one)

Local (free): Install Ollama → run ollama serve → run ollama pull gemma2:9b. Aether auto-detects your models.

Cloud (fast): Get a free Groq API key → paste in Settings (⌘ ,) → done.

3

Connect a Project

Click the folder icon or use File → Open Folder to connect your project. The file explorer and AI get full codebase access.

4

Chat with AI

Click the ✨ sparkle icon to open AI chat. Ask questions, request code changes, or run the quick-start prompts. The AI reads files, writes patches, and runs commands — always asking for approval.

5

AI Auto-Delegates Complex Tasks

For multi-file features, refactoring, or anything complex — the AI automatically delegates to the 8-agent team. No mode toggle needed. Agent progress and approvals show inline in chat.
